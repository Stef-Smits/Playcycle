# Playcycle Website

**Playcycle** is a sustainable, modular game controller that users can repair and customize themselves.

This website is a prototype that includes:

- **Landing Page** – Introduces Playcycle, its benefits, and an email waitlist for early access.
- **Webshop** – Browse controllers and replacement parts like joysticks, buttons, and triggers.
- **Product Pages** – Individual pages with product details, prices, and installation instructions.
- **Production Process Page** – Explains how the controllers are designed, sourced, and assembled sustainably.

The website is built using **HTML, CSS, and Bootstrap** and is fully responsive.

> ⚠️ This is a fictional prototype created for educational purposes. No real purchases or services are processed.
